# Eventify — Event Discovery Platform

Eventify is a modern, responsive, and high-fidelity landing page for discovering upcoming events. It features a stunning "Refined Editorial" aesthetic with dynamic animations, modular components, and real-time search.

## Features

- **Single-File Delivery**: The entire project is consolidated into a single `index.html` file, including all CSS, JavaScript, and event data.
- **Dynamic Aurora Background**: A custom HTML5 Canvas background that generates a smooth, interactive "aurora" effect.
- **Floating Particles**: Subtle micro-animations using a custom particle system for an immersive feel.
- **Live Search**: Instantly filter events as you type without page reloads.
- **Responsive Design**: Fully optimized for Desktop, Tablet, and Mobile devices with a custom drawer menu for mobile users.
- **Aesthetic Excellence**: Built with a "Refined Editorial Light" theme, featuring:
  - Playfair Display serif headings.
  - DM Sans body typography.
  - Vibrant but sophisticated color palette.
  - Glassmorphic navigation and card effects.

## File Structure

The project has been optimized for simplicity:
- **`index.html`**: The complete application source. No external dependencies are required other than Google Fonts.

## How to Use

1. Simply open the `index.html` file in any web browser.
2. Browse the featured events or use the search bar to find specific experiences.
3. Click "Register" to see a success interaction.
4. Interact with the page to see the background follow your mouse.

## Technical Details

- **Vanilla CSS**: Advanced styling using modern CSS variables, grid layouts, and custom keyframes.
- **Vanilla JavaScript**: Lightweight and fast logic without heavy frameworks.
- **Data-Driven UI**: Events are dynamically generated from a central JSON-like structure within the code, making it easy to update or extend.
