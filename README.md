## 🎨 Overview

**Eventify** is a single-page event discovery platform that demonstrates high-end web design and complex motion graphics using only **HTML5, CSS3, and Vanilla JavaScript**.

Designed with a "Refined Editorial Light" aesthetic, it prioritizes typography, spatial balance, and fluid interactions to create a luxurious software-as-a-service (SaaS) or premium consumer feel.

## ✨ Key Features

*   **Dynamic Motion Background**: A custom HTML5 `<canvas>` rendering a slow-moving, mouse-reactive Aurora borealis effect, layered beneath a subtle grain texture and floating CSS physics particles.
*   **Deep Glassmorphism UI**: Extensive use of `backdrop-filter` for frosted glass effects on navigation bars, search inputs, and event cards, complete with responsive hover lifts and shadow depths.
*   **Micro-Interactions**: Features such as staggered entrance animations on load, sweep-shimmer effects on primary buttons, and dynamic color-swapping on secondary actions.
*   **Live Event Filtering**: Instantaneous, client-side search functionality filtering a JavaScript event array.
*   **Zero Dependencies**: The entire project is self-contained in a single file, relying purely on native web technologies (aside from Google Fonts CSS). No React, Vue, or Tailwind — just vanilla mastery.

## 🛠️ Technology Stack

*   **Structure**: Semantic HTML5
*   **Styling**: Vanilla CSS3 (Custom Properties/Variables, Flexbox, Grid, Animations/Keyframes, Backdrop Filters)
*   **Logic**: Vanilla JavaScript (ES6+ Classes, DOM Manipulation, requestAnimationFrame)
*   **Typography**: Playfair Display (Headings) & DM Sans (UI/Body) from Google Fonts

## 🚀 Getting Started

1.  Clone this repository.
2.  Open `index.html` in any modern web browser.
3.  No build steps, no package managers. It just runs.

## 💡 Design System Notes

*   **Color Palette**: Grounded in soft creams (`#FAFAF8`) with a striking burnt orange accent (`#D4622A`) to draw attention to primary actions.
*   **Typography Scale**: A fluid font scale ensuring sharp readability across mobile, tablet, and widescreen viewports using `clamp()`.
